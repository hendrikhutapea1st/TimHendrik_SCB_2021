Deskripsi Perusahaan

PayPal adalah platform yang memberi fitur kepada pengguna untuk mengelola uang dan menawarkan pilihan yang fleksibel saat mengirim, membayar, atau menerima pembayaran dengan transaksi yang aman dan mudah tanpa menggunakan kartu kredit atau debit.

PayPal awalnya didirikan oleh Max Levchin , Peter Thiel , dan Luke Nosek pada Desember 1998 sebagai perusahaan yang mengembangkan perangkat lunak keamanan untuk perangkat genggam. Sebenarnya, nama asli perusahaan itu adalah Confinity.

Alasan PayPal Menggunakan Node.js

Node.js membantu menyelesaikan masalah batas antara browser dan server dengan mengaktifkan aplikasi browser dan server untuk ditulis dalam JavaScript. Ini menyatukan spesialisasi satu tim yang memungkinkan kami untuk memahami dan bereaksi terhadap kebutuhan pengguna kami di tingkat mana pun dalam tumpukan teknologi.

- Jeff Harrel, Direktur Senior Produk dan Teknik Pembayaran di PayPal

Open Source Repository

Node.js telah menyediakan berbagai macam modul yang sudah jadi dan siap digunakan dan tentu saja itu gratis

Cross Platform JavaScript Runtime Environment

Dengan kata lain, Node.js dapat bekerja dengan JavaScript yang memungkinkan pengembang menguji dan men-debug kode mereka

Scalable Systems

Node.js ringan dan memudahkan untuk menskalakan aplikasi dengan menerapkan instance ke server yang berbeda atau menggunakan modul built-in cluster yang melakukan fork proses ke semua inti CPU

Integrated API

Rata rata aplikasi atau web memerlukan komunikasi backend dengan database atau layanan eksternal. Dengan Node.js, ketika membutuhkan library untuk membuat REST atau API

Realtime Apps

Node.js menangani banyak operasi I/O yang membuka pintu untuk aplikasi kehidupan nyata. Baik itu messenger, obrolan, pelacakan makanan, pelacakan taksi, alat kolaborasi, dan lain sebagainya

Reusable Code

Kode dapat dengan mudah digunakan kembali antara komponen yang berbeda atau antara backend dan frontend

Response Time

Node.js memberikan 35% penurunan waktu respons rata-rata untuk halaman yang sama. Ini menghasilkan halaman yang disajikan 200 md lebih cepat sesuatu yang pasti akan diperhatikan pengguna
